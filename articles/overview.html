Overview

This project implements a client side router using the following technologies:

  • History API and popstate event
  • Web Components
  • View Transitions API (single-document)

Notice that View Transitions API support is not mandatory. Everything will just work, without transitions.

The goal is to use the router on GitHub Pages. Checkout how to setup the router.

The API is declarative, via ghp-* custom elements. Checkout how to use the router.
